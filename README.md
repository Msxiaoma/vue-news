# 使用 vue 仿今日头条app

```
该项目仅用于学习项目，没有搭建后台，使用 mock + axios 模拟异步请求,使用 vuex 组件之间共享数据

```

## 技术栈

1. html + css + js 
2. vue
3. vuex
4. mock + axios

## 实现功能

1. 首页展示
2. 查看消息
3. 选项卡切换不同新闻
4. 点击选项卡的 + 按钮，实现频道的添加和删除
5. 点击搜索按钮，输入关键字，回车进行实时搜索，在结果中高亮显示关键字
6. 图文懒加载
7. 点击导航栏的刷新按钮只实现了按钮的旋转特效，并没有实现页面刷新加载暑假班

## 项目部署

1. 安装依赖 npm install
2. 开发环境 npm run dev      # serve with hot reload at localhost:8080
3. 生产环境 npm run build
